# Руководство по настройке на Android

Vibe Tavern работает локально на Android через Termux + proot Ubuntu. APK управляет всем, но при первой настройке нужно выполнить несколько шагов вручную.

## Требования

- **Android-устройство** с процессором ARM64
- **Termux** установленный из [F-Droid](https://f-droid.org/packages/com.termux/) (**НЕ** из Play Store — та версия сломана и заброшена)
- ~500 МБ свободного места для контейнера Ubuntu + Vibe Tavern

## Первичная настройка

Эти шаги нужно выполнить **один раз** перед тем, как APK начнёт работать. После этого всё работает в один тап.

### Шаг 1: Установи Termux из F-Droid

1. Открой [F-Droid](https://f-droid.org/packages/com.termux/) на телефоне.
2. Скачай и установи Termux.
3. Открой Termux и дождись завершения загрузки.

### Шаг 2: Обнови пакеты Termux

Свежая установка Termux часто содержит устаревшие пакеты, из-за которых ломается `curl` и другие инструменты. Выполни **внутри Termux**:

```
apt update && apt full-upgrade
```

Нажми `y` и Enter при запросе. Это может занять минуту.

> **Важно:** Если видишь предупреждение "No mirror selected", сначала выполни `termux-change-repo`, выбери зеркало и повтори команду выше.

### Шаг 3: Разреши внешним приложениям выполнять команды

APK нужен доступ к выполнению команд в Termux. Выполни **внутри Termux**:

```
mkdir -p ~/.termux
echo "allow-external-apps=true" >> ~/.termux/termux.properties
termux-reload-settings
```

### Шаг 4: Перезапусти Termux

Закрой Termux полностью:
1. Набери `exit` в Termux и нажми Enter.
2. Смахни Termux из списка недавних приложений.
3. Открой Termux заново из меню приложений.

Это нужно, чтобы настройка `allow-external-apps` вступила в силу.

### Шаг 5: Установи APK Vibe Tavern

1. Скачай APK Vibe Tavern из [Releases](https://github.com/Noineri/vibe_tavern/releases) и установи.
2. Открой приложение.
3. Если ты пропустил(а) какой-то из шагов 1–4, APK обнаружит проблему и подскажет что не хватает.
4. Выдай Android-разрешение: **"Run commands in Termux environment"** — приложение само запросит.
5. Нажми **📦 Установить / обновить**.

APK сделает всё сам:
- Скопирует сборку Vibe Tavern на устройство
- Установит пакеты Termux (curl, tar, proot-distro, procps)
- Настроит контейнер proot Ubuntu
- Распакует Vibe Tavern в `~/vibe-tavern/` внутри Ubuntu
- Запустит сервер

Это занимает 1–2 минуты. Прогресс видно в окне Termux.

## Ежедневное использование

### Запуск сервера

Нажми **🚀 Запустить сервер** в APK. Откроется видимая Termux-сессия с диагностическим логом, затем запустится сервер.

Когда заработает — нажми **🌐 Открыть в браузере** — откроется `http://127.0.0.1:8787` в браузере телефона.

> **Совет:** Не закрывай Termux пока пользуешься Vibe Tavern. Если смахнёшь из недавних — сервер остановится.

### Остановка сервера

Нажми **⏹ Остановить сервер** в APK. Или через уведомление "Server is running" → "Stop Server".

### Обновление

Нажми **🔄 Обновить программу** (та же кнопка, меняет название после первой установки). Переустановит сборку поверх существующей. Твои данные (чаты, персонажи, настройки) сохраняются — они лежат в отдельной директории.

### Удаление

Два варианта в APK:

- **Удалить Vibe Tavern** — удаляет файлы программы, чаты, настройки. Контейнер Ubuntu остаётся на случай переустановки.
- **Удалить всё** — удаляет весь контейнер proot Ubuntu целиком.

## Решение проблем

### "CANNOT LINK EXECUTABLE curl" / ошибки SSL

Пакеты Termux устарели. Открой Termux и выполни:

```
apt update && apt full-upgrade
```

Затем перезапусти Termux (exit → смахнуть → открыть заново) и попробуй снова.

### Кнопки в APK ничего не делают

Проверь по порядку:

1. **Termux из F-Droid?** Версия из Play Store сломана. Удали её и установи из [F-Droid](https://f-droid.org/packages/com.termux/).
2. **Разрешил(а) внешние приложения?** Открой Termux и выполни:
   ```
   mkdir -p ~/.termux
   echo "allow-external-apps=true" >> ~/.termux/termux.properties
   termux-reload-settings
   ```
   Затем **полностью перезапусти Termux** (exit → смахнуть → открыть заново).
3. **Выдал(а) разрешение?** APK запрашивает "Run commands in Termux environment". Проверь: Настройки Android → Приложения → Vibe Tavern → Разрешения.
4. **Обновил(а) пакеты?** Выполни `apt update && apt full-upgrade` в Termux.

### Скрипт установки показывает "No mirror selected"

Открой Termux и выполни:

```
termux-change-repo
```

Выбери зеркало (обычно подходит по умолчанию), затем retry установку через APK.

### Веб-интерфейс лагает или зависает

- Отключи оптимизацию батареи для Termux в настройках Android.
- Держи Termux в списке недавних приложений (не смахивай).
- Отключи агрессивный энергосберегающий режим если он есть на телефоне.

### Браузер не открывается

- Открой вручную: перейди по адресу `http://127.0.0.1:8787` в любом браузере.

### Ошибка "proot-distro not found"

- Нужно сначала установить. Нажми **📦 Установить / обновить**.

## Как это работает

APK — это **оркестратор сервера**, а не веб-клиент. Он:

1. Управляет контейнером proot Ubuntu внутри Termux
2. Запускает ARM64-бинарник Vibe Tavern внутри этого контейнера
3. Открывает браузер телефона по адресу `http://127.0.0.1:8787`

Никакого WebView, никакого встроенного рендеринга. Браузер обрабатывает всё — клавиатуру, прокрутку, рендеринг. Это сделано намеренно: нативный браузер быстрее и надёжнее любого встроенного WebView.

Сборка Vibe Tavern внутри APK — это предкомпилированный ARM64-бинарник с встроенным фронтендом. Никакого `git clone`, `bun install` или сборки на устройстве.

### Расположение файлов (внутри proot Ubuntu)

| Путь | Содержимое |
|------|-----------|
| `~/vibe-tavern/` | Файлы программы (бинарник, веб-ассеты, миграции) |
| `~/.local/share/vibe-tavern/` | Данные пользователя (база, сводки, ассеты) |
| `~/start-vibe-tavern.sh` | Стартовый скрипт с переменными окружения |

### Логи

| Лог | Расположение (в домашней директории Termux) |
|-----|---------------------------------------------|
| Лог установки | `~/vibe-tavern-install.log` |
| Лог запуска | `~/vibe-tavern-start.log` |
| Лог остановки | `~/vibe-tavern-stop.log` |
| Лог удаления | `~/vibe-tavern-uninstall.log` |
