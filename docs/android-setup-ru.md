# Руководство по настройке на Android

Vibe Tavern работает локально на Android через Termux + proot Ubuntu. APK управляет всем — не нужно вводить команды вручную.

## Требования

- **Android-устройство** с процессором ARM64
- **Termux** установленный из [F-Droid](https://f-droid.org/packages/com.termux/) (НЕ версия из Play Store — она сломана)
- ~500MB свободного места для контейнера Ubuntu + Vibe Tavern

## Установка

1. Скачать и установить APK Vibe Tavern из [Releases](https://github.com/Noineri/vibe_tavern/releases).
2. Открыть приложение. Если Termux не установлен, приложение направит тебя на F-Droid.
3. Выдать Android-разрешение: **"Run commands in Termux environment"** — приложение само запросит.
4. Нажать **📦 Установить / обновить**.

APK сделает всё сам:

- Скопирует сборку Vibe Tavern на устройство
- Установит пакеты Termux (curl, tar, proot-distro, procps)
- Настроит контейнер proot Ubuntu
- Распакует Vibe Tavern в `~/vibe-tavern/` внутри Ubuntu
- Запустит сервер

Это занимает минуту-две. Прогресс видно в окне Termux.

## Запуск сервера

Нажать **🚀 Запустить сервер в Termux** в APK. Откроется видимая Termux-сессия с 6-шаговым диагностическим логом, затем запустится сервер.

Когда заработает — нажать **🌐 Открыть в браузере** — откроется `http://127.0.0.1:8787` в браузере телефона.

> **Совет:** Не закрывайте Termux пока пользуетесь Vibe Tavern. Если смахнуть из недавних — сервер остановится.

## Остановка сервера

Нажать **⏹ Остановить сервер** в APK. Или через уведомление "Server is running" → "Stop Server".

## Обновление

Нажать **🔄 Обновить программу** (та же кнопка, меняет название после первой установки). Переустановит bundled-сборку поверх существующей. Твои данные (чаты, персонажи, настройки) сохраняются — они лежат в отдельной директории.

## Удаление

Два варианта в APK:

- **Удалить Vibe Tavern** — удаляет файлы программы, чаты, настройки. Контейнер Ubuntu остаётся на случай переустановки.
- **Удалить всё** — удаляет весь контейнер proot Ubuntu целиком.

## Решение проблем

### Кнопка «Запустить» ничего не делает

- Проверь Termux-сессию на наличие ошибок.
- Убедись что Termux установлен из F-Droid, а не из Play Store.
- Убедись что выдал разрешение **"Run commands in Termux environment"**.
- В Termux проверь что `~/.termux/termux.properties` содержит `allow-external-apps=true`. Установщик пытается прописать это автоматически, но некоторые версии Android это блокируют.

### Веб-интерфейс лагает или зависает

- Отключи оптимизацию батареи для Termux в настройках Android.
- Держи Termux в списке недавних приложений (не смахивай).
- Отключи агрессивный энергосберегающий режим если он есть на телефоне.

### Браузер не открывается

- Открой вручную: перейди по адресу `http://127.0.0.1:8787` в любом браузере.

### Ошибка "proot-distro not found"

- Нужно сначала установить. Нажми **📦 Установить / обновить**.

## Как это работает

APK — это **оркестратор сервера**, а не веб-клиент. Он:

1. Управляет контейнером proot Ubuntu внутри Termux
2. Запускает ARM64-бинарник Vibe Tavern внутри этого контейнера
3. Открывает браузер телефона по адресу `http://127.0.0.1:8787`

Никакого WebView, никакого встроенного рендеринга. Браузер обрабатывает всё — клавиатуру, прокрутку, рендеринг. Это сделано намеренно: нативный браузер быстрее и надёжнее любого встроенного WebView.

Сборка Vibe Tavern внутри APK — это предкомпилированный ARM64-бинарник с встроенным фронтендом. Никакого `git clone`, `bun install` или сборки на устройстве.

### Расположение файлов (внутри proot Ubuntu)

| Путь                          | Содержимое                                       |
| ----------------------------- | ------------------------------------------------ |
| `~/vibe-tavern/`              | Файлы программы (бинарник, веб-ассеты, миграции) |
| `~/.local/share/vibe-tavern/` | Данные пользователя (база, сводки, ассеты)       |
| `~/start-vibe-tavern.sh`      | Стартовый скрипт с переменными окружения         |

### Логи

| Лог           | Расположение (в домашней директории Termux) |
| ------------- | ------------------------------------------- |
| Лог установки | `~/vibe-tavern-install.log`                 |
| Лог запуска   | `~/vibe-tavern-start.log`                   |
| Лог остановки | `~/vibe-tavern-stop.log`                    |
| Лог удаления  | `~/vibe-tavern-uninstall.log`               |
