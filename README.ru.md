## Что такое Vibe Tavern?

Vibe Tavern — это локальное веб-приложение написанное на [Bun](https://bun.sh) + [React](https://react.dev) для AI-ролплея, где можно создавать персонажей, управлять лором, собирать промпты и общаться через любой OpenAI-совместимый API. С UI, который не заставляет разбираться в нагромождении настроек.

> **Статус:** 🛠️ Beta-2. Базовый функционал ролплея стабилен и протестирован. [Что пока не готово →](#что-пока-не-готово)

## Главные фичи

### 💬 Чат

- **Ничего лишнего** - только вы и персонаж. Переключайте чаты, персон, пресеты промптов и AI-модели из быстрого доступа, не отвлекаясь от общения.
- **Стриминг с reasoning** — блоки размышлений свернуты по умолчанию, чтобы не мешать основному процессу чата. Впрочем, вы можете их открыть и увидеть, что размышления выглядят стильно.
- **Свайпы вариантов ответа** — десктопная анимация слайдов + мобильная карусель из трех панелей. Встроен подсчет токенов и лейбл модели к конкретному свайпу.
- **Подсчет токенов прямо в UI** — следи за бюджетом контекста без открытия настроек, прямо в окне чата! Реализована цветовая индикация, а также всплывающее окно, где показано, что сколько занимает.
- **Стильная цветовая палитра** - оптимизирована для долгого чтения, по крайней мере темная тема. Кто вообще использует светлую тему?

### 🧠 Встроенная генерация сводок

- **Визуальный фидбек по токенам** — "Без сводки: 12,753t → Сэкономлено 11,814t (93%)".
- **Авто-сводка** — фоновая генерация сводок по принципу "настроил и забыл" каждые N сообщений.
- **Диапазонные сводки** — каждая охватывает определенный диапазон сообщений, можно включать/отключать по отдельности.
- **Управление количеством сообщений в контексте** - модель может смотреть только последние N сообщений, а остальной контекст использовать для чего-то еще.

### ✍️ Редактор на CodeMirror

- Полнофункциональный редактор JS-скриптов с подсветкой синтаксиса, поиском и нумерацией строк.
- Слишком сложно писать скрипты самому? Используй шаблон! Не подходит шаблон? ЗАСТАВЬ встроенного ИИ-помощника переписать его!

### 🔍 Prompt Tracer

- Вы можете увидеть как собрался промпт, слой за слоем.
- Количество токенов на слой, глубина инъекций, источник активации.

### 📦 Совместимость с SillyTavern

- Импорт карточек персонажей (V2/V3 PNG).
- Импорт лорбуков, истории чатов, пресетов промптов.
- Массовое сканирование директорий — укажи папку с ST и импортируй всё разом.
- Полная поддержка макросов (macro) из ST (`{{user}}`, `{{char}}`, `{{if}}`, `{{setvar}}`, `{{roll}}` и т.д.).

### 📱 Доступ с мобильного

- Открой **Mobile Access** в веб-интерфейсе и сканируй QR-код → общайся с телефона в той же LAN-сети или через Tailscale/VPN.
- QR/copy URL содержит актуальный access token в `#token=...`; браузер сохраняет его локально и отправляет с API-запросами.
- Удалённый API работает fail-closed: LAN/Tailscale-клиентам нужен актуальный token, а локальный `127.0.0.1` остаётся без пароля.
- Опциональный TLS.
- Android ARM64 билд с Termux-оркестратором.

### 🛠️ Для продвинутых юзеров

- **Движок лорбуков** — активация по ключевым словам, логика AND/OR/NOT, cooldown, веса групп, рекурсия, глубина сканирования, WI якоря.
- **Скрипты** — песочница `node:vm` с совместимым с Janitor AI API.
- **Движок макросов** — парсер рекурсивного спуска на базе AST, вложенные блоки `{{if}}`, переменные, броски кубиков.
- **5 типов провайдеров** — OpenAI-compat (например OpenRouter, DeepSeek, Groq, xAI и др.), Anthropic, Google, Ollama, llama.cpp.
- **Пресеты промптов** — полный контроль над system prompt, jailbreak, author's note и кастомными инъекциями с указанием глубины.

## Быстрый старт

### Windows

#### Вариант 1:

Скачай и запусти последний `.exe` из [Releases](https://github.com/Noineri/vibe_tavern/releases), Vibe Tavern откроется в твоем браузере.

> [!NOTE]
> Исполняемый файл пока экспериментальный. Он работает, но требует больше тестов. Если что-то сломалось, попробуй способ с `.bat` ниже:

#### Вариант 2:

Скачай из [Releases](https://github.com/Noineri/vibe_tavern/releases) zip архив, распакуй и запусти файл `Start RP Platform.bat`

### Linux / macOS

```bash
git clone https://github.com/Noineri/vibe_tavern.git
cd vibe_tavern
bash ./Vibe_Tavern.sh
```

Скрипт `Vibe_Tavern.sh` проверит обновления, сделает бэкап директории data и запустит Vibe Tavern

### Docker

```bash
docker compose up -d
```

### Android (Termux)

APK билд с оркестратором под Termux — автоматизирует большую часть настройки, но всё равно нужно будет выдать доступы. Обратись к [Гайду по установке на Android](docs/android-setup-ru.md).

## Что пока не готово

Vibe Tavern находится в стадии Beta-2. Вот чего пока не хватает по сравнению со взрослыми аналогами:

- **Система плагинов/расширений** — пока нет сторонних расширений.
- **Групповые чаты** — только один персонаж + твоя персона.
- **Генерация изображений** — без интеграции с A1111/ComfyUI.
- **TTS / STT** — нет поддержки голоса.
- **Vector/RAG** — нет поиска на основе эмбеддингов (но обычная активация лорбуков работает).
- **Мало тестов на краевые кейсы** — основные флоу протестированы, но необычные комбинации могут что-то сломать.

Если наткнулся на баг, [заводи issue](https://github.com/Noineri/vibe_tavern/issues). Это очень поможет.

## Стек технологий

| Слой        | Технология                                                                              |
| ----------- | --------------------------------------------------------------------------------------- |
| Runtime     | [Bun](https://bun.sh) — быстрый "всё в одном" JavaScript, TypeScript и JSX тулкит       |
| Backend     | [Hono](https://hono.dev/) — type-safe RPC, 14KB                                         |
| Frontend    | [React 19](https://react.dev) + [Tailwind CSS 4](https://tailwindcss.com)               |
| База данных | SQLite через [Drizzle ORM](https://orm.drizzle.team)                                    |
| AI Стриминг | [Vercel AI SDK](https://ai-sdk.dev/) — унифицированный интерфейс для разных провайдеров |
| Язык        | TypeScript                                                                              |

**Структура монорепозитория:**

```
vibe_tavern/
├── packages/domain/          # Типы, ID, константы (без зависимостей)
├── packages/api-contracts/   # Zod-схемы, общие для фронта и бэка
├── packages/db/              # Drizzle сторы
├── packages/prompt-pipeline/ # Чистая функция сборки, движок макросов
├── packages/import-export/   # Парсеры карточек и чатов
├── services/api/             # Бэкенд (Hono + выполнение AI-запросов)
└── apps/web/                 # Фронтенд SPA (React 19)
```
